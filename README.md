# CommonOS

CommonOS is a deployment and management framework for persistent AI agent fleets.

Most agents today run as short-lived processes — stateless, isolated, and difficult to operate at scale. CommonOS changes that by giving every agent its own isolated cloud sandbox: a dedicated computer, runtime, and persistent filesystem.

On top of these per-agent environments, CommonOS provides a shared control plane for provisioning, task routing, permissions, monitoring, and coordination across the fleet. A live spatial interface makes it easy to see which agents are running, what they're doing, and how work is progressing across the system in real time.

The result is infrastructure that turns agents from transient executions into persistent digital workers — more reliable, more secure, and practical to run together as a single managed system.
